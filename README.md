# Customer Churn Analysis Dashboard

## Project Overview
This Power BI dashboard analyzes customer churn patterns and retention behavior across different customer segments. It provides a clear view of total customers, active and inactive members, retained customers, and exited customers.

## Objective
The main goal of this dashboard is to identify churn trends and understand which customer categories are more likely to stay or leave. It helps turn raw customer data into useful business insights.

## Key Metrics
- Total Customers
- Credit Card Holders
- Non Credit Card Holders
- Active Customers
- Inactive Customers
- Exited Customers
- Retained Customers

## Visuals Used
- Monthly customer analysis by active and inactive category
- Retain vs exit trend analysis
- Gender-wise customer distribution
- Bank DOJ by credit type analysis

## Tools Used
- Power BI
- SQL
- Excel
- Data Cleaning
- Data Visualization

## Dashboard Insights
- The dashboard shows overall customer distribution and churn status.
- Monthly trends help identify how active and inactive customers change over time.
- Gender and credit type breakdowns provide segmentation insights.
- Retained customers are higher than exited customers, which helps evaluate customer stability.
